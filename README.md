# RoadToJavascript

> Javascript school

## Installation

The easiest way is to keep `roadtojavascript` as a devDependency in your `package.json`.
```json
{
  "devDependencies": {
    "roadtojavascript": "0.0.1"
  }
}
```

You can simple do it by:
```bash
npm install roadtojavascript --save-dev
```
